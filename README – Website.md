## Introducere

Am lucrat la acest proiect in cadrul facultatii cu scopul de a invata cum se creea un website de la 0.
Am invatat partea de afisare a CSS-ului si a lucrurilor afisate prin HTML pana la partea de animatii si butoane folosind Javascript si
modificare a bazelor de date din partea de site.

Tot website-ul si baza de date din spate sunt impreuna intr-un container din Docker Desktop.


---
## Cum este gandit
Tema site-ului sunt donatiile. Am gandit un sistem format din useri si administratori. Sistemul este gandit ca userii sa isi poata crea donatii liber, avand un buton special de "Creaza Donatie" care apasat
va deschide un pop-up ce va cere numele, o descriere, o poza si o suma initiala. De asemenea am creat si o pagina web destinata evenimentelor unde userii isi pot crea de asemenea 
evenimente, sa introduca titlul, descrierea, locatia si data si durata evenimentului. Toate donatiile si evenimentele create de useri si modificarile 
pe care le aduc asupra lor precum donatii de sume de bani vor aparea de asemenea si in baza de date.
Am gandit de asemenea si un al doilea tip de utilizator cu rol de admin, acesta putand si sterge slot-urile de donatii si evenimente. 

De asemenea am creat si pagini cu diferite roluri: pagina web de creare de cont unde userul isi introduce numele, prenumele, email si parola, si o pagina de contact cu un formular in care utilizatorul isi 
exprima nemultumirea. Fiecare cont creat si nemultumire trimisa de utilizatori vor ajunge in baza de date in tabelul speicific
